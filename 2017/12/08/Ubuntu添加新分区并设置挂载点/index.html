Ubuntu 添加新分区 并设置挂载点

发表于 2017-12-08 | 分类于 树莓派 ， Ubuntu | | 阅读次数

最近入手了一块树莓派，安装了Ubuntu mate，sd卡还剩7g没有分区，添加新分区 并设置挂载点

1、创建新分区

首先查看已经有的分区：

一共16g存储可以看出还有7个多g空间没分配，已有/dev/mmcblk0p1、/dev/mmcblk0p2两个主分区，那么我们可以创建一个/dev/mmcblk0p3
可以到dev下面查看设备(串口SATA硬盘是 /dev/sda 如果是IDE的硬盘 即 并口的 那就是 /dev/hda如果不知道自己到底是什么类型的 可以cd /dev 然后ls sda之类找下有没有sda 或者hda) ，本文用的sd卡

输入m查看帮助

输入n添加分区

输入p选择主分区

输入3，选择分区号

敲击回车选择默认开始分区位置

敲击回车选择默认结束分区位置

输入w保存，会自动退出

重启
